Module Forward_torch.Maths

Contains all the maths operators that automate batched forward-mode differentation.

operations on the Maths.t object. Maths.t object is a dual number of (primal, tangent) where the primal is of type Tensor.t and the tangent is of type tangent.

type tangent =
  1. | Direct of Torch.Tensor.t
  2. | Lazy of unit -> Torch.Tensor.t
type t = Torch.Tensor.t * tangent Stdlib.Option.t
